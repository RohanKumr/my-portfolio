Overview

Arrivals is an e-commerce website I built to explore clean coding practices and improve my frontend development skills. This project highlights my ability to integrate third-party APIs, manage state efficiently using Redux and Middleware library (Saga), and build feature-rich, dynamic web pages.

Key Features

  • Home Page: Includes product sliders, featured collections, and product listings.
  • Product Page: Displays detailed product descriptions, related products, and options to add items to the cart.
  • Login System: Allows users to log in for a personalized shopping experience.
  • Cart Management: Users can easily add items to the cart and proceed to checkout.
  • Responsive Design: Works seamlessly on both mobile and desktop devices.

What I Did

  • Built the website using Next.js for fast and responsive pages.
  • Managed app state using Redux and handled side effects with Saga middleware.
  • Fetched and displayed product data dynamically using the Fake Store API.
  • Focused on writing clean and reusable code to make the project easy to maintain.

What I Learned

  • How to integrate third-party APIs for real-time data fetching.
  • Managing state and asynchronous tasks efficiently using Redux and Saga.
  • Designing an intuitive and responsive user interface for an e-commerce platform.
  • The key components of building an online store, such as cataloging, cart functionality, and user authentication.

Arrivals was a great learning experience, helping me understand how to build and manage a functional e-commerce platform while maintaining clean and organized code.
