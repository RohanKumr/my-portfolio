Overview

Bubbles is a fun and interactive game I created while learning the fundamentals of game development in JavaScript. This project allowed me to explore key programming concepts and improve my skills in animation, collision detection, and dynamic gameplay design.

What I Did

  • Developed a Class-Based Game: Utilized JavaScript classes to structure the game logic, keeping the code modular and organized.
  • Implemented Collision Detection: Designed a system to detect interactions between bubbles and fish, creating a dynamic and engaging gameplay experience.
  • Used HTML Canvas: Leveraged the HTML canvas element to render the gameâ€™s visuals and animations.

Gameplay Highlights

  • Popping the Bubble: Players score points by successfully popping bubbles while avoiding the fish.
  • Dynamic Gameplay: Integrated a scoring system and designed the game to increase in speed as time progresses, adding a challenging element to the gameplay.
  • Animation and Interaction: Added smooth animations and reactive elements to make the game visually appealing and responsive.

What I Learned

  • Core Game Development Concepts: Strengthened my understanding of object-oriented programming, particularly class-based design in JavaScript.
  • Collision Detection: Gained experience in implementing logic to handle interactions between multiple objects on the screen.
  • Canvas Animations: Learned how to use the HTML canvas to create dynamic and interactive visuals.
  • Balancing Difficulty: Understood the importance of designing a game with progressively increasing difficulty to keep players engaged.

This project was a significant step in my journey into gaming and web development. It taught me essential skills and gave me the confidence to tackle more complex interactive projects.
