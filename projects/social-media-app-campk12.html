Overview:

Developed a social media platform prototype as part of an interview assignment for Camp K12. This platform showcased the integration of essential social media features, focusing on usability.

What I Did:

  • User Management: Created functionalities for user registration, login, and authentication with form validation for a secure user experience.
  • Dynamic Feeds: Implemented a feature to fetch and display user-generated posts in a real-time feed.
  • Interactive Features: Added emoji and GIF support to enhance user engagement while posting updates.
  • Profile Pages: Designed individual user profiles showcasing their feeds and other relevant details.
  • Infinite Scrolling: Built infinite loading functionality for feeds to create a seamless browsing experience.
